## Sistema de Gerenciamento da CAA - Comércio Amazonense de Alumínio

O sitema de genrenciamento da CAA tem o objetivo de adicionar novas rotinas ao winthor como:

- Desmembramento de pedido.
- Alteração de pedido.
- Desbloqueio de pedido.

## Regras de Acesso

Um usuário só podera ter acesso ao sistema se houver uma permissão prévia feita a rotina criada.
